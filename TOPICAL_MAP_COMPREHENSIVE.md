## INTERNAL LINKING STRATEGY

### Hub-Spoke Linking Pattern

**Primary Hub Pages (50+ internal links each)**
- Link to 8+ relevant calculators contextually
- Link to 5+ secondary hub guides  
- Link to 3+ comparison pages
- Link to 2+ tax/advanced guides
- Include breadcrumb navigation
- Related content section with next steps

**Secondary Hub Pages (30–40 internal links each)**
- Link to primary hub (beginning and end)
- Link to 3+ supporting guides
- Link to 2+ calculators
- Link to reference pages
- Link to 1+ comparison page
- Link to 1+ related niche guide

**Tertiary Pages (8–15 internal links each)**
- Link to primary hub (early)
- Link to relevant secondary hub
- Link to 2–3 related calculators
- Link to 2–3 related guides
- "Related Topics" section with 3–5 links

### Entity Co-Occurrence Rules
Every page should mention:
1. **Primary entity** (page topic) — 15+ times naturally
2. **2–3 related entities** (Tier 2) — 5–10 times each
3. **Contextual entities** (Tier 3–4) — 2–5 times each

### Cross-Cluster Linking
- Revenue diversification → Hub links to specific stream hubs
- Growth discussion → Growth Hub + Monetization Hub
- Tax implications → Advanced Hub + Monetization Hub
- Multi-platform → Multi-Platform Hub + specific platform hubs
